# Design System

This example shows how to build a simple design system with renoun and Next.js.

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/souporserious/renoun/tree/main/examples/design-system&project-name=renoun-design-system&repository-name=renoun-design-system)

## How to use

Use the [`create-renoun`](https://github.com/souporserious/renoun/tree/main/packages/create-renoun) CLI to copy this example to your machine:

```bash
npx create-renoun --example design-system
```
